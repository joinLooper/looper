## Known limitations and remaining blocker

Actual LINE/LIFF credential QA remains the sole release blocker. Until it is
completed, authenticated Player browser scenarios—including terminal-state refresh,
knowledge-card UI persistence, logout, and real multi-account isolation—are backed
by full API/runtime tests and frontend helper tests, but not by a real LIFF browser
session. This build must not be described as multi-user beta ready, closed beta
ready, production ready, or security complete.
