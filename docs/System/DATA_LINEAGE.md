# 數據牽動關係

## 任務接取

- 新增：任務接取紀錄
- 增加：待核銷數量
- 不改變：星星、能量、經驗值、現金

## 店家核銷

- 減少：待核銷數量
- 增加：完成任務數量
- 新增：星星帳本、能量帳本、核銷紀錄、審計紀錄
- 可能增加：經驗值帳本（升等規則確定後）
- 不改變：真實現金

## 星星消耗

- 新增：負數星星帳本
- 減少：星星餘額
- 不改變：能量、經驗值、等級、現金

## 能量消耗

- 新增：負數能量帳本
- 減少：能量餘額
- 可能改變：遊戲世界狀態
- 不改變：星星、經驗值、現金

## 現金交易

- 新增：金流交易與金流帳本
- 可能改變：平台實收、退款、店家應付款
- 不得自動改變：星星、能量、經驗值、等級

## 玩家升等

- 來源：累積經驗值達到門檻
- 改變：玩家等級、解鎖內容
- 不扣除：經驗值
- 不改變：現金、星星、能量餘額
